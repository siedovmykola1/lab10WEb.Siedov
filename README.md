Blazor Todo List Application
Опис
Цей проєкт — це простий вебзастосунок "Список справ", створений на базі ASP.NET Core Blazor в межах лабораторної роботи №10.

Функціонал
Додавання нових завдань до списку.

Візуальне позначення виконаних завдань.

Підрахунок кількості активних справ.

Технології
.NET 8.0

Blazor (Server/WebAssembly)

C# / Razor Components
